# CrawlerTennis 当前保留功能

这个仓库目前只保留 3 类能力：

- 查询所有网球场馆的可预约时段
- 查询江湾室外体育场-网球的可预约时段
- 定时检查并取消即将开始的预约

预约点击、并发抢票、按优先级自动预约、按指定时间预约、历史实验脚本，已经从仓库中清理。

## 1. 统一入口

统一入口脚本是：

```bash
./tennis_craw.sh
```

当前只支持以下子命令：

```bash
./tennis_craw.sh query-tennis [--json] [--headed] [--only-available]
./tennis_craw.sh query-jw [--json] [--headed] [--only-available]
./tennis_craw.sh cancel-soon [提前量] [--headed]
./tennis_craw.sh cancel-loop [提前量] [检查间隔] [--headed]
./tennis_craw.sh examples
```

## 2. 功能说明

### `query-tennis`

查询当前所有网球场馆的可预约时段。

示例：

```bash
./tennis_craw.sh query-tennis
./tennis_craw.sh query-tennis --only-available
./tennis_craw.sh query-tennis --json
./tennis_craw.sh query-tennis --headed
```

说明：

- 默认输出所有网球场馆
- `--only-available` 只保留当前存在可预约时段的场馆和日期
- `--json` 输出原始结构化结果

对应实现：

- Python：`query_jiangwan_slots.py`
- Playwright：`tests/query_jiangwan_slots.spec.ts`
- 查询 helper：`tests/helpers/tennis_availability.ts`

### `query-jw`

只查询“江湾室外体育场-网球”的可预约时段。

示例：

```bash
./tennis_craw.sh query-jw
./tennis_craw.sh query-jw --only-available
./tennis_craw.sh query-jw --json
./tennis_craw.sh query-jw --headed
```

说明：

- 内部仍复用全场馆查询逻辑
- 但输出会自动收敛成江湾室外体育场-网球的日期列表
- `--json` 时输出的是江湾场馆的 `days` 数组，不是全场馆数组

### `cancel-soon`

取消距离开始时间落在指定提前量窗口内的预约。

示例：

```bash
./tennis_craw.sh cancel-soon
./tennis_craw.sh cancel-soon 20m
./tennis_craw.sh cancel-soon 1h
./tennis_craw.sh cancel-soon 90s
```

说明：

- 默认提前量是 `60s`
- 支持 `s`、`m`、`h`
- 会先读取当前预约列表，再筛出即将开始的记录，并逐条执行取消

对应实现：

- Python：`cancel_scheduler.py`
- 预约读取/取消封装：`reservation_client.py`
- Playwright：`tests/cancel_dry.spec.ts`、`tests/cancel_by_time.spec.ts`
- 预约 helper：`tests/helpers/reservation.ts`

### `cancel-loop`

按固定间隔重复执行 `cancel-soon`。

示例：

```bash
./tennis_craw.sh cancel-loop
./tennis_craw.sh cancel-loop 11m 2m
./tennis_craw.sh cancel-loop 1h 10m
```

说明：

- 默认提前量是 `60s`
- 默认检查间隔是 `1h`
- 适合挂在长期运行的会话里做简单轮询

对应实现：

- Python：`cancel_scheduler_loop.py`

## 3. 直接调用方式

不走壳脚本时，可以直接运行 Python：

```bash
python query_jiangwan_slots.py
python query_jiangwan_slots.py --only-available
python query_jiangwan_slots.py --jiangwan-only
python cancel_scheduler.py 20m
python cancel_scheduler_loop.py 20m 5m
```

## 5. 运行前提

当前代码默认依赖以下环境：

- Python 3.10+
- Node.js
- 已安装 npm 依赖
- 已安装 Playwright Chromium
- 运行机器可以访问 `https://booking.fudan.edu.cn`

常见准备命令：

```bash
npm install
npx playwright install chromium
```

说明：

- 当前脚本默认调用的是 `python`，不是 `python3`
- 代码使用了 Python 3.10+ 的类型注解语法
- 如果你的 `python3` 还是 3.9，不建议直接用 `python3` 跑这些脚本
